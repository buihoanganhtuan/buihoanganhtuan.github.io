LTE/NR Resources
Ever wondered how your 4G/5G smartphone can access Internet anytime and anywhere? Let's find out how a smartphone can connect wirelessly to the ubiquitous access points known as "base stations".
Computer Networking Resources
Once the data packets from your smartphone reach a base station and then a router, they will start an incredible journey through interconnected networks until they either reach their destinations, or get dropped midway. Want to know how all of that happen?
Web Development Resources
If you are browsing a website, and your request packets have successfully reached the server hosting the site, the server will send back necessary files to tell the web browser how to display the requested site and how to react when receiving your input. Want to learn how to create such files in details?
Data Structures and Algorithms
In order to provide us with the best possible experiences, the computers, be it your smartphone, the base stations, the severs, or even the routers, need organized ways (structures) to store data in their memory and logics (algorithms) that make use of them in the most efficient way. Are you ready to dive into the exiciting world of data structures and algorithms?
